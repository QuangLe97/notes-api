## API

**GET ALL NOTES**
* METHOD : GET
* URL : host:8787/notes
* PARAMS :
  * page : int
  * limit: int
  * < name field >.< [contains,equals,in] >: str  ex: name.contains
  * sort :str
  
**GET DETAIL NOTE**
* METHOD : GET
* URL : host:8787/notes/ID

**CREATE NOTE**
* METHOD : POST
* URL : host:8787/notes
* BODY : {"name":str, "content":str}

**UPDATE NOTE**
* METHOD : PUT
* URL : host:8787/notes/ID
* BODY : {"name":str, "content":str}

**DELETE NOTE**
* METHOD : DELETE
* URL : host:8787/notes/ID
